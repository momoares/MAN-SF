# CCBDA Final Project - MAN-SF Implementation

## Environment Setup
- Pytorch=1.8.1
- Python=3.8
- Tensorflow=2.3.0
- `pip install -r requirements.txt`

## Run
- First, do data preprocessing, `python data_preprocess.py`
	- option `--train_size`
- After doing date preprocessing, you can run it by `python train.py`
